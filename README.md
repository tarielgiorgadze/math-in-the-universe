# 🌌 Math in the Universe

An interactive Python application that explores how mathematics appears in **space, nature, and scientific precision**.

## 🎯 Modules

| Module | Description |
|--------|-------------|
| **Pi and Space Precision** | Animated visualization showing why using π = 3.14 instead of real π can cause navigation errors at interplanetary distances |
| **Monte Carlo Pi Simulation** | Estimate π using random points inside a circle – watch convergence in real time |
| **Golden Ratio and Nature** | Fibonacci spiral, golden rectangles, and phyllotaxis (sunflower-like pattern) animated together |

## 🖥️ Technologies

- Python 3
- Tkinter (menu GUI)
- Matplotlib (animations & visualizations)
- NumPy (calculations)
